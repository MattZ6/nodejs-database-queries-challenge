## 🎉 Começando

Clone o repositório:

```bash
git clone https://github.com/MattZ6/nodejs-database-queries-challenge
```

Entre na pasta do projeto:
```bash
cd nodejs-database-queries-challenge
```

Instale as dependências:

```bash
yarn
```

## 🧪 Testes

Para executar os testes:

```bash
yarn test
```
